# Image Generation

## Skill Role

这是公用的图片生成/编辑能力，负责把用户的视觉创作需求交给底层图片模型完成。它可以从文字生成图片，也可以基于参考图做普通编辑、风格化、重绘、创意改造或画幅重构。

## Use When

- 用户用纯文字要求生成一张新图片、海报、插画、场景图、产品图或概念图。
- 用户发送一张参考图，并要求改风格、重绘、优化、替换局部视觉元素、增强氛围。
- 用户要求把单张图重新构图、适配某种画幅、改变比例或延展画面内容。
- 用户希望保留参考图主体和风格，但生成一张新的变体。

## Do Not Use When

- 用户只想理解、评价、描述或比较图片，不需要新图；使用 `image_understanding`。
- 用户只是聊天、咨询、写作或文本翻译；使用 `general`。
- 用户只是要求幽默吐槽或娱乐回应；使用 `funny`。
- 用户明确要求处理非图片文件，且有更具体的文件 skill 可用。

## Editing Policy

- 普通单图小修默认保留原图画布、比例和尺寸，避免无意裁切或变形。
- 如果用户明确要求重构画幅、适配手机端、改比例、延展背景或重新构图，应让图片模型按目标画幅生成，不要在保存阶段强行恢复原图尺寸。
- 多图输入默认理解为参考合成或创意融合，但不要承诺精确像素级控制。
- 用户要求越具体，提示词越应该保留原意并补足构图、光线、色彩、质感、风格和质量要求。

## Output Contract

成功时返回 `image_path` 和简短说明；失败时返回可读的失败原因。不要引用与本能力无关的专用能力名称。
